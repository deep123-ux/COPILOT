# 🚀 Scenario Based Copilot

A web-based AI assistant platform where users can create customized bots based on specific scenarios and metadata. Each bot can answer queries using the Gemini API.

## 🌟 Features

* Create AI bots based on scenarios
* Add metadata to guide responses
* Live image preview for each bot
* Attractive UI with hero banner
* Bot dashboard with cards and images
* Chat interface for interacting with bots
* Data stored using LocalStorage
* Integration with Gemini 2.5 Flash model

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript
* Gemini API
